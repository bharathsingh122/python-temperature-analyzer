# python-temperature-analyzer
A beginner-friendly Python project that calculates and analyzes temperature data using Python fundamentals such as variables, loops, conditions, and functions.



temperatures = [30, 32, 28, 35, 31, 29, 33]

average = sum(temperatures) / len(temperatures)
highest = max(temperatures)
lowest = min(temperatures)

print("Temperature Analysis")
print("-------------------")
print("Average Temperature:", average)
print("Highest Temperature:", highest)
print("Lowest Temperature:", lowest)
